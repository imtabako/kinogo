# Install
## Client Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

## Server Setup
```sh
npm install
```
### Start server

```sh
npm run start
```
